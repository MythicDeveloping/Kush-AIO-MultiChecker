# Features
- [x] Cui & Log Checking Modes
- [x] Update Checks
- [x] Config File
- [x] Save Remaining Lines
- [x] Multichecking
- [x] In-app Settings
- [x] Automatic Proxy Rotation
- [x] Auth Proxy Support
- [x] Captcha Solving Support (2Captcha,AntiCaptcha,AnyCaptcha)
